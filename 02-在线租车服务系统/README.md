# 02-在线租车服务系统
在线租车服务系统是一个以java+mysql进行开发的java租车管理系统，系统还是一个基于租车业务场景而搭建的O2O服务平台，可为用户提供商务租车、接送机、旅游租车、企业租车、自驾租车、婚庆用车等自助租车服务。
在线租车服务系统包含车辆库管理、门店管理、员工管理、司机管理、订单管理、活动管理、评价管理、财务管理、统计等。
