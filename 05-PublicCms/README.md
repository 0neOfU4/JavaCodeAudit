# 05-PublicCms
PublicCMS是采用2021年最新主流技术开发的开源JAVACMS系统。

200多万行代码修改 反复打磨 现代化java cms，轻松支撑千万数据、千万PV；支持静态化，服务器端包含; 目前已经拥有全球0.0003%的用户，语言支持中、繁、日、英；是一个已走向海外的成熟CMS产品
